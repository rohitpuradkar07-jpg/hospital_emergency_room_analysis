# Hospital Emergency Room Dashboard

An interactive **Power BI dashboard** built to analyze hospital emergency room performance and patient trends using real ER visit data.


## Overview

This project focuses on turning raw emergency room data into clear, actionable insights through data visualization and dashboard design.  
It helps track key operational metrics such as patient volume, average wait time, satisfaction score, and department referrals.

The dashboard was built entirely in **Power BI**.

## Project Objectives

- Monitor emergency room performance over time
- Analyze patient wait times and satisfaction levels
- Identify referral patterns by department
- Understand patient demographics such as age, gender, and race
- Support data-driven decisions for hospital operations

## Dataset Summary

- **Total Records:** 9,216
- **Date Range:** April 2023 to October 2024
- **Key Fields:**
  - Patient ID
  - Admission Date
  - Gender
  - Age
  - Race
  - Department Referral
  - Admission Flag
  - Satisfaction Score
  - Wait Time

## Key Metrics Tracked

- **Patient Volume**
- **Average Wait Time**
- **Patient Satisfaction Score**
- **Department Referrals**

## Dashboard Pages

### 1. Monthly View
Provides month-by-month analysis of ER performance and patient trends.

### 2. Consolidated View
Shows overall performance across a selected date range for a broader operational view.

### 3. Patient Details
Displays patient-level information for deeper analysis and case-level review.

### 4. Key Takeaways
Summarizes major patterns, anomalies, and actionable insights from the data.

## Visual Insights Included

- Admitted vs. Non-Admitted patients
- Patient age distribution
- Department referral trends
- Percentage seen within 30 minutes
- Gender and racial demographics
- Patient volume by day and hour

## Tools Used

- **Power BI**
- **DAX**
- **Data Modeling**
- **Data Cleaning**
- **Dashboard Design**

## Key Findings

- The average patient wait time is approximately **35.3 minutes**
- The average satisfaction score is approximately **4.99 out of 10**
- Patient flow and referral patterns reveal opportunities for operational improvement

## Project Goal

The goal of this dashboard is to provide a clear, professional view of ER performance so hospital teams can improve efficiency, reduce wait times, and enhance patient experience.
